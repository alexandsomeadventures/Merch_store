Sylvie Discord

I am currently developing Sylvie, a sophisticated language model for Discord, leveraging the capabilities of both the Gemini API and the Disnake API. Sylvie is designed to efficiently respond to user requests within Discord servers, offering a seamless and interactive experience. The ultimate goal is to transform Sylvie into an autonomous AI agent capable of managing and enhancing the user experience in Discord communities, making it a valuable tool for server administrators and members alike.

ChildChat

ChildChat is a fully functional social media app designed for children, developed using pure programmatic UIKit and Swift. It features engaging animations, secure authentication, and a smooth UI/UX experience. The app includes an efficient search algorithm for finding events, RSVP functionality, incoming user notifications, and comprehensive profile management. Remarkably, ChildChat was built in just 2 days and earned me an internship as a SwiftUI intern at Arizona State University (ASU), demonstrating rapid development and effective problem-solving skills while providing a safe and enjoyable platform for young users to interact and participate in various activities.
